### spec
- HTML5, CSS, js
- 크로스브라우징 : Chrome, firefox, edge, safari
- 미디어쿼리 : 작은 화면 지원 (break point: ~ 420px)
- JOSN 데이터 활용
- `input`, `select` 로 필터링 기능 추가
- '더보기' 버튼 클릭 시 데이터 추가
- '리셋' 버튼으로 기존 데이터 제거하고 새로운 데이터를 랜덤으로 추가
